# @sxo/theme-antd

SXO 设计系统的 Ant Design 风格主题。

## 安装

```bash
pnpm add @sxo/theme-antd @sxo/engine @sxo/design
```

## 用法

```typescript
import { createSxo } from 'vue-sxo';
import antdTheme from '@sxo/theme-antd';

app.use(createSxo({
    tokens: antdTheme
}));
```

## 主题特色
- **专业感**: 基于 Ant Design 5.0 设计规范。
- **平衡配色**: 经典的技术蓝作为主色。
