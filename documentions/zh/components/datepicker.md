# DatePicker 日期选择器

当用户需要选择一个日期时。

## 何时使用

- 需要选择特定的日期、月份或年份。
- 提供直观的日历面板交互。

## API


### API

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| modelValue | 绑定值 | `[Date` | `-` |
| placeholder | 占位提示语 | `String` | `"Select date"` |
| size | 尺寸 | `String` | `"md"` |
| variant | 变体样式 | `String` | `"outline"` |
| rounded | 是否圆角 | `Boolean` | `true` |
| format | 格式化字符串 | `String` | `"YYYY-MM-DD"` |
